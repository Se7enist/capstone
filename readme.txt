The training_model.ipynb file trains the model using the california_accidents.csv dataset. 
The trained model is saved in the accident_severity_model.pkl file. accident_monitor_sf.py is the main program for the San Francisco accident monitoring system. 
To start the system, enter streamlit+run+filename in the console. 
sf_accidents.csv stores historical data for the San Francisco area.
kl_accident_monitor.py is the accident monitoring system for the Kuala Lumpur area, 
and kl_accidents.csv stores historical data for the Kuala Lumpur area.
